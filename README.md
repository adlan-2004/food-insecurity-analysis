# Food Insecurity & Student Health Analysis (Econometrics)

## Project Overview
This project applies econometric analysis to examine the relationship between food insecurity and health outcomes among undergraduate students at IIUM.

Using primary survey data (n = 852), the study investigates how socio-economic factors, lifestyle behaviors, and financial support influence:
- Self-rated health (SRH)
- Body Mass Index (BMI)
- Depressive symptoms
- Food insecurity levels

The analysis aims to provide data-driven insights into student well-being and financial vulnerability.

---

## Tools & Methods
- Econometric Modeling (OLS Regression)
- EViews
- Statistical Testing (VIF, White Test, Breusch-Pagan)
- Survey Data Analysis

---

## Methodology
Multiple regression models were developed to analyze relationships between variables:

- Health Model (SRH)
- BMI Model
- Depressive Symptoms Model
- Food Insecurity Model

Key variables include:
- Food insecurity score
- Household income per capita
- Financial comfort
- Physical activity
- Education level
- Zakat and scholarship support
- Environmental shocks (e.g. floods)
